# Privacy Policy for N-Styler

**Last Updated: January 7, 2026**

## Overview

N-Styler is a browser extension that allows users to inject custom CSS and JavaScript into websites. We are committed to protecting your privacy.

## Data Collection

**N-Styler does NOT collect, store, or transmit any personal data.**

All data created by users (custom CSS/JS code, settings, and preferences) is stored locally in your browser using Chrome's Storage API. This data never leaves your device.

## Data Storage

The following data is stored locally on your device:
- Custom CSS and JavaScript code you create
- Domain-specific rules and settings
- Language preferences
- External file paths you configure

This data is stored using `chrome.storage.local` and remains entirely on your device.

## Data Sharing

We do not:
- Collect any personal information
- Track your browsing activity
- Send any data to external servers
- Share any information with third parties
- Use analytics or tracking tools

## Permissions

N-Styler requires certain browser permissions to function:

| Permission | Purpose |
|------------|---------|
| `storage` | Save your custom CSS/JS code locally |
| `activeTab` | Access current tab URL to apply domain-specific rules |
| `scripting` | Inject your custom CSS/JS into web pages |
| `tabs` | Identify which domain rules to apply |
| `<all_urls>` | Apply your styles to any website you choose |

## External Files

If you choose to load external CSS/JS files, those files are fetched directly by your browser. N-Styler does not proxy, cache, or monitor these requests.

## Changes to This Policy

We may update this Privacy Policy from time to time. Any changes will be reflected in the "Last Updated" date above.

## Contact

If you have questions about this Privacy Policy, please create an issue on our GitHub repository.

# 개인정보 처리방침 (N-Styler)

**최종 수정일: 2026년 1월 7일**

## 개요

N-Styler는 사용자가 웹사이트에 커스텀 CSS와 JavaScript를 적용할 수 있는 브라우저 확장 프로그램입니다. 우리는 사용자의 개인정보 보호를 최우선으로 합니다.

## 데이터 수집

**N-Styler는 어떠한 개인 데이터도 수집, 저장, 전송하지 않습니다.**

사용자가 생성한 모든 데이터(커스텀 CSS/JS 코드, 설정, 환경설정)는 Chrome Storage API를 사용하여 브라우저에 로컬로 저장됩니다. 이 데이터는 절대 사용자의 기기를 벗어나지 않습니다.

## 데이터 저장

다음 데이터가 사용자의 기기에 로컬로 저장됩니다:
- 사용자가 작성한 커스텀 CSS 및 JavaScript 코드
- 도메인별 규칙 및 설정
- 언어 설정
- 사용자가 설정한 외부 파일 경로

이 데이터는 `chrome.storage.local`을 사용하여 저장되며 전적으로 사용자의 기기에 남아있습니다.

## 데이터 공유

우리는 다음을 하지 않습니다:
- 어떠한 개인정보도 수집하지 않음
- 브라우징 활동을 추적하지 않음
- 외부 서버로 데이터를 전송하지 않음
- 제3자와 정보를 공유하지 않음
- 분석 또는 추적 도구를 사용하지 않음

## 권한

N-Styler는 기능 수행을 위해 특정 브라우저 권한이 필요합니다:

| 권한 | 목적 |
|------|------|
| `storage` | 커스텀 CSS/JS 코드를 로컬에 저장 |
| `activeTab` | 도메인별 규칙 적용을 위해 현재 탭 URL에 접근 |
| `scripting` | 웹 페이지에 커스텀 CSS/JS 적용 |
| `tabs` | 적용할 도메인 규칙 식별 |
| `<all_urls>` | 사용자가 선택한 모든 웹사이트에 스타일 적용 |

## 외부 파일

외부 CSS/JS 파일을 로드하도록 선택한 경우, 해당 파일은 브라우저가 직접 가져옵니다. N-Styler는 이러한 요청을 프록시하거나 캐시하거나 모니터링하지 않습니다.
