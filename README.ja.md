# N-Styler

[English](README.en.md) | [한국어](README.md) | **日本語**

ウェブページにカスタムCSSとJavaScriptを注入できるChrome拡張機能です。

## 主な機能

- **ドメインベースのスタイル/スクリプト管理**: ドメイン別にCSSとJSを登録し、自動的に適用
- **外部ファイルサポート**: ローカルまたはリモートのCSS/JSファイルパスを登録可能
- **ページリソース管理**: ページに読み込まれたCSSリソースの確認とトグル
- **リアルタイム適用**: ページ更新なしで即座に適用
- **多言語対応**: 英語、日本語、韓国語UI対応
- **簡単管理**: 直感的なポップアップUIで簡単に管理

## URLマッチング方式

**ドメイン(ホスト名)ベース**のマッチングを使用します:
- `example.com`のすべてのページに同じスタイルを適用
- サブドメイン区別可能 (`www.example.com` ≠ `api.example.com`)
- クエリパラメータやパスの影響を受けない

## インストール

### 開発者モード（ローカル）
1. Chromeで `chrome://extensions` にアクセス
2. 右上の「開発者モード」を有効化
3. 「パッケージ化されていない拡張機能を読み込む」をクリック
4. `dist` フォルダを選択

### Chrome ウェブストア
- [Chrome ウェブストアからインストール](#) (リリース後にリンクを更新予定)

## 使用方法

1. 拡張機能アイコンをクリック
2. **CSS/JSタブ**: カスタムコードを入力
3. **Externalタブ**: 外部ファイルパスを登録
4. **Resourcesタブ**: ページのCSSリソースをトグル
5. 「Save」ボタンをクリック - 即座に適用！

## フォルダ構造

```
N-Styler/
├── src/                    # 開発ソースコード
│   ├── _locales/           # 多言語対応 (en, ja, ko)
│   ├── background/         # バックグラウンドスクリプト
│   ├── content/            # コンテンツスクリプト
│   ├── icons/              # 拡張機能アイコン
│   ├── popup/              # ポップアップUI
│   └── manifest.json       # 拡張機能設定
├── dist/                   # 配布ファイル
├── README.md               # 韓国語ドキュメント
├── README.en.md            # 英語ドキュメント
├── README.ja.md            # 日本語ドキュメント
├── TODO.md                 # 開発計画
├── store-description.md    # ストア説明
└── N-Styler.zip            # 配布ZIPファイル
```

## プライバシー

- すべてのデータはブラウザのローカルに保存されます
- 外部サーバーへのデータ送信はありません
